# How To Install BloodPengu

Works on:

- Windows 11
- Kali Linux (Debian Based)
- And more

### Git Clone

Copy the GitHub Repository:

```
git clone https://github.com/pengu-apm/BloodPengu.git
```

Run installation

```
bash install.sh
Installing BloodPengu...
[+] Copying files...
[+] Building...
[+] Creating launcher...
Installation complete!

run: bloodpengu
```
